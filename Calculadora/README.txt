INSTRUÇÕES pt-br:

1 - Baixe o aplicativo NETBEANS https://netbeans.apache.org/download/nb124/nb124.html ;
  1.1 -  Escolha a versão referente ao seu SO;
  1.2 -  Caso seu SO seja 64bits baixe o aplicativo -- Apache-NetBeans-12.4-bin-windows-x64.exe;
  1.3 -  Siga as instruções disponibilizadas pelo próprio instalador e conclua a instalação.

2 - Baixe o aplicativo JDK https://www.oracle.com/br/java/technologies/javase-jdk11-downloads.html ;
  2.1 -  Escolha a versão referente ao seu SO;
  2.2 -  Que provavelmente será a ||  Windows x64 Installer - 151.83 MB;
  2.3 -  Siga as instruções disponibilizadas pelo próprio instalador e conclua a instalação.

3 - Abrir o aplicativo JAVA;
  3.1 - Baixe a pasta calculadora disponibilizada no meu github;
  3.2 - Após coloque ela no diretório C:\Users\Administrador\Documents\NetBeansProjects que é o repositório padrão do JAVA NetBeans;
  3.3 - Abra o NETBEANS e vá em -- File / Open Project -- Automaticamente a calculadora estará lá, agora basta selecionar ela e clicar em -- Open Project;
  3.4 - Para faze-lá funcionar aperte SHIFT + F6 e aguarde.
