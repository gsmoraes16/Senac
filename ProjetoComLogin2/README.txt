INSTRUÇÕES pt-br:

1 - Baixe o aplicativo chamado "xampp" --> https://www.apachefriends.org/download.html ;
  1.1 -  Escolha a versão referente ao seu SO;
  1.2 -  De preferência a versão mais atualizada disponivel, atualmente é a 8.0.7;
  1.3 -  Siga as instruções disponibilizadas pelo próprio instalador e conclua a instalação.
  
  1.4 - Configurar o "xampp";
    1.4.1 - Após a instalação abra o programa, 
            caso não ache ou não lembre onde foi instalado pressione a tecla "Windows" e pesquise "xampp",
	    após isso irá localizar o aplicativo, o aplicativo que deve ser aberto é o "XAMPP Control Panel";
    1.4.2 - Agora basta clicar em "start" no "module" - MySQL, após isso o nome deve ficar verde, oque significa que está funcionando;

2 - Instalação do BD (Banco de Dados);
  2.1 - Baixe o arquivo --> https://www.mediafire.com/file/vwq435yl6dim5vi/projetosnc.sql/file ,pronto, o banco foi instalado, agora vamos para importação dele;

3 - Importação do BD (Banco de Dados)
  3.1 - Abre o seu navegador de preferencia e digite na barra de pesquisa: http://localhost:8080/phpmyadmin/ ;
  3.2 - Na parte superior do site clique em "Importar" indentificado por um papel com um seta vermelha;
  3.3 - Pronto! Agora clique em "escolher arquivo", selecione o arquivo que foi baixado anteriormente, normalmente ele vai estar na pasta "downloads" e é chamado de "mvc".

INSTRUCTIONS en-un:

1 - Download the application called "xampp" --> https://www.apachefriends.org/download.html ;
  1.1 - Choose the version referring to your OS;
  1.2 - Preferably the most updated version available, currently it is 8.0.7;
  1.3 - Follow the instructions provided by the installer and complete the installation.
  
  1.4 - Configure "xampp";
    1.4.1 - After installation open the program,
            if you don't find or don't remember where it was installed press the "Windows" key and search for "xampp",
            after that it will find the application, the application that should be opened is the "XAMPP Control Panel";
    1.4.2 - Now just click "start" in "module" - MySQL, after that the name should be green, which means it's working;

2 - Installation of DB (Database);
  2.1 - Download the file --> https://www.mediafire.com/file/vwq435yl6dim5vi/projetosnc.sql/file, ready, the database was installed, now we're going to import it;

3 - BD Import (Database)
  3.1 - Open your favorite browser and type in the search bar: http://localhost:8080/phpmyadmin/ ;
  3.2 - At the top of the site click on "Import" identified by a paper with a red arrow;
  3.3 - Ready! Now click on "choose file", select the file that was previously downloaded, normally it will be in the "downloads" folder and is called "mvc".
